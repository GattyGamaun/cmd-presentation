Command Prompt

On the screen you can see how command prompt looks like. To open cmd navigate to search line, type cmd and press enter. If you make right click on the main bar, you see some properties which you can change, like font, font size color, etc.

1. Listing files in the directory

dir

The first command you should to know is dir. Dir shows subdirectories and the quantity of the folders at the bottom.

2. Change a drive

Simple command to change drive. Type label of the drive and than type colon. Drive is changed.

3. Moving into a directory

cd {path}\{directory name}

The third command is cd. It helps navigate throw directories. If you type cd dot dot, you move up on the one level. One dot means the same level of the tree structure. To navigate to subfolder, type cd and the folder name. If you don't remember exactly name of the folder, type dir to see subdirectories. Also you can specify the path to specific folder. Path can be absolute or relative.

4.1. Make a directory

mkdir {directory name}

In the CMD to make a folder is pretty simple. Type mkdir with the folder name.

4.2. Make subdirectories

mkdir {path}\{directory name}

If you want to make inserted folders specify the path.

5. Rename a directory

ren {old name} {new name}

Ren allows to rename the name of the folder. You have to specify which folder you are going to rename then write the new name.

6. Remove a directory

rd {directory name}

To remove the folder type rd, what means remove directory, and type the name.

7. Copy a file

copy {source} {destination}

To copy a file you have to specify the path from place where file is located and then the path to new place. Uncomfortable, isn't it?

8. Create a file

copy con {file name}

Copy also suitable for creating files. Type copy con with new file name and press enter. If you are going to make text file you can type words which will be inserted into the file, or leave it blank. Press ctrl+z, then enter. You'll see the message that file is copied that means the file was created.

9. Copy folders

xcopy /s {source} {destination}

Copy folders is awful operation. Type xcopy, slash, s, then specify source folder and destination path. Notice that only subfolders will be copied. /s means subdirectories will be included.

10.1. Delete all files in the folder

del {folder which consists files}

Del helps to delete all files which placed in the folder. This command can not delete a folder.

10.2. Delete a file

del {file name}

If you want to delete certain file just type the file name with del command. If you are not in the current folder specify the path to file.

11. Lunch an application

Just type the name of the application with {file name}

Sometimes, while you are searching files you need to open it. Type name of the app which can open this file and the file name. If app can open folders type app name and the dot that means open all included files.

12. All commands are available in the help

If you forget a command or want to explore new commands just type help to view all possible commands.
