UNIX shell

On the screen you can see how shell looks like. To open cmd navigate to search line, type cmd and press enter. If you make right click on the main bar, you see some properties which you can change, like font, font size color, etc.

1.1 Listing files in the directory

ls

The first command you should to know is ls. lists all files and directories in the working directory.

1.2 Listing files in the directory

ls -a

lists all contents of a directory, including hidden files and directories

1.3 Listing files in the directory

ls -alt

Multiple options can be used together, like ls -alt

2 Show current directory

pwd

pwd outputs the name of the current working directory

3.1 Moving into a directory

cd {path}/{directory name}

The third command is cd. It helps navigate throw directories. If you type cd dot dot, you move up on the one level. One dot means the same level of the tree structure. To navigate to subfolder, type cd and the folder name. If you don't remember exactly name of the folder, type dir to see subdirectories. Also you can specify the path to specific folder. Path can be absolute or relative.

3.2 Moving into home directory

cd [Enter]

If you type cd and press the Return key—with no directory specified—you’ll go back to your Home folder. (You can also type cd ~ to go there.)

4 Make a directory

mkdir {directory name}

In the shell to make a folder is pretty simple. Type mkdir with the folder name.

5 Make a file

touch {file name}

To make a file use touch command

6.1 Remove a file

rm {file name}

To remove the file type rm, what means remove file, and type the name.

6.2 Remove a directory

rm -r {directory name}

To remove the folder type rm -r, what means remove directory, and type the name.

6.3 Delete directories with subdirectories

rm -r {folder}

rm -r helps to delete all subfolders which placed in the folder.

7 Copy a file

copy {source/file} {destination}

To copy a file you have to specify the path from place where file is located and then the path to new place.

8.1 Move a file

mv {source}{file name} {destination}

If you want to move certain file type mv with source of the file name with name, then specify new location.

8.2 Rename a file

mv {old file name} {new file name}

To rename file type mv, old name and then new name of the file.

9 Clear display

clear

Clear clears window

10 Lunch an application

Just type the name of the application with {file name}

Sometimes, while you are searching files you need to open it. Type name of the app which can open this file and the file name. If app can open folders type app name and the dot that means open all included files.

11 Edit file

nano {file name}

nano opens editor like vim editor

12 How to know about installed processor

winsat cpu -v

This command help to know which processor you computer has. It works for Windows.

13 A part of commands are available in the help

If you forget a command or want to explore new commands just type help to view all possible commands.
